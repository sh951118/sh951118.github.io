      기존의 방식 람다 방식
      반환타입 메소드명 (매개변수, ...) {
        실행문
      }       		(매개변수, ... ) -> { 실행문 ... }

  • Java 12

    • 개선된 switch문 표현방식 지원

      • switch문에서 "->" 사용으로, 코딩 간소화

                    기존 문법
                String testSwitch = "";
                switch(data){
                    case "one" :
                        testSwitch = "1";
                        break;
                    case "two" :
                        testSwitch = "2";
                        break;
                    default:
                }

                람다식 문법
                String testSwitch = "";
                switch(data){
                    case "one" -> testSwitch = "1";
                    case "two" -> testSwitch = "2";
                    default -> throw new Exception ~~
                }
